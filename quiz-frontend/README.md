# Quiz App Frontend

A React-based frontend application for the Quiz App microservices backend.

## Features

### Admin Features
- **Create Quiz**: Create new quizzes by specifying title, category, and number of questions
- **Add Question**: Add questions to the question bank with multiple choice options

### User Features
- **Take Quiz**: Take quizzes by entering a Quiz ID
- **View Score**: Get immediate feedback on quiz performance with percentage and score

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Backend services running:
  - API Gateway (port 8083)
  - Question Service (port 8080)
  - Quiz Service (port 8087)
  - Service Registry

## Installation

1. Navigate to the frontend directory:
```bash
cd quiz-frontend
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## Usage

### For Admins

1. **Login as Admin**:
   - Go to the login page
   - Enter your name
   - Select "Admin" role
   - Click Login

2. **Create a Quiz**:
   - Navigate to "Create Quiz"
   - Enter quiz title
   - Enter category (e.g., Java, Python, JavaScript)
   - Enter number of questions
   - Click "Create Quiz"
   - Note the Quiz ID that will be created (you may need to check the backend logs or database)

3. **Add Questions**:
   - Navigate to "Add Question"
   - Fill in the question details:
     - Question text
     - Four options
     - Correct answer (must match one of the options)
     - Difficulty level
     - Category
   - Click "Add Question"

### For Users

1. **Login as User**:
   - Go to the login page
   - Enter your name
   - Select "User" role
   - Click Login

2. **Take a Quiz**:
   - Navigate to "Available Quizzes"
   - Click "Start Quiz by ID"
   - Enter the Quiz ID provided by your administrator
   - Answer all questions
   - Click "Submit Quiz"
   - View your score

## API Configuration

The frontend is configured to communicate with the API Gateway at `http://localhost:8083`. 

The API Gateway routes requests to:
- `/question-service/*` - Question Service endpoints
- `/quiz-service/*` - Quiz Service endpoints

## Technologies Used

- React 19.2.3
- React Router DOM 7.11.0
- Axios 1.13.2
- CSS3 (with modern gradients and animations)

## Notes

- The application uses localStorage to persist user sessions
- Quiz IDs are generated by the backend when a quiz is created
- All API calls go through the API Gateway
- CORS is configured in the API Gateway to allow requests from `http://localhost:3000`

## Troubleshooting

1. **Cannot connect to backend**:
   - Ensure all backend services are running
   - Check that the API Gateway is accessible at `http://localhost:8083`
   - Verify CORS configuration in the API Gateway

2. **Quiz not found**:
   - Verify the Quiz ID is correct
   - Ensure the quiz was created successfully in the backend

3. **Questions not loading**:
   - Check that questions exist in the database for the specified category
   - Verify the Question Service is running and accessible
